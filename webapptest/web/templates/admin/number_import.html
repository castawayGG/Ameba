🔌 Авто-импорт номеров

Загружает список номеров из внешнего HTTP-источника и добавляет их как неактивные аккаунты. Уже существующие номера пропускаются.

Настройки источника

Значение заголовка Authorization. Оставьте пустым, если не нужен.

Путь через точку к массиву номеров в JSON-ответе. Например: data.phones

Результат импорта

📖 Поддерживаемые форматы

  • Custom JSON API — эндпоинт возвращает JSON-массив строк / объектов с полем phone/number/tel.
  • Remote CSV / TXT URL — файл с номерами в первой колонке (по одному на строку).

Пример JSON-ответа:

{"data": {"phones": ["+79001234567", "+79007654321"]}}

JSON path для примера выше: data.phones
